上机几步走：

uptime  看1，5，15分钟的负载

free -h  看内存还剩多少

df -h  看一下磁盘使用率

ps aux   看进程，如果查问题的话还可以配合grep

ss -tnlp  看端口



整合成shell脚本：

#!/bin/bash
echo "========== 系统健康检查报告 =========="
echo ">>> 当前时间：$(date)"
echo ""
echo ">>> 系统负载："
uptime
echo ""
echo ">>> 内存使用情况："
free -h
echo ""
echo ">>> 磁盘使用情况："
df -h | grep -v tmpfs
echo ""
echo ">>> 进程状态："
ps aux
echo ""
echo ">>> 监听端口："
ss -tlnp | grep -E ':(80|443)'



刚刚用的远程ssh跑df -h的效果可能不明显，用上面那个脚本在本地跑可以非常直白的看出效果
